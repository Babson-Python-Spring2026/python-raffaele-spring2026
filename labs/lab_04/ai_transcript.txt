PHASE 2 - BYTECODE INSPECTION
================================================================

USER: Now I want to confirm my guesses against the actual bytecode.
Please explain how to disassemble a .pyc file in Python without
running it.

AGENT: The .pyc format is: 16-byte header, then a marshalled code
object. So:

    import marshal
    with open('game_obfuscated.pyc', 'rb') as f:
        f.read(16)                # skip header
        top = marshal.load(f)
    import dis
    dis.dis(top)

Importantly, `dis.dis(top)` recurses into every code object nested
inside `top.co_consts`, so you see the bytecode for every function
the module defines, even if those functions have meaningless names
like `_5`, `_6`, `_c` after obfuscation.

But disassembly can be noisy. A more targeted move: walk the code
objects and print, for each one, the tuple of co_names (globals and
attributes it touches) and the string/int constants in co_consts.
That gives you a skeleton of what each function does without
reading a single bytecode instruction.

  _c:  touches _2, _3; takes 2 args (_d, _e); constants [0]

USER: OK, I want to write the reconstruction myself. Give me the

AGENT: Skeleton:

  STATE (module globals or passed around)
    height, width, num_mines
    mines : set of (row, col)
    revealed : 2d list initialized to HIDDEN ('v' or diamond)

  FUNCTIONS
    clear_screen()
    get_board_settings() -> (h, w, m)       [prompts + clamp]
    place_mines(h, w, m) -> set             [random.randrange loop]
    in_bounds(r, c, h, w) -> bool
    is_mine(r, c, mines) -> bool
    count_neighbors(r, c, h, w, mines) -> int
    display_board(revealed, h, w, reveal_all=False, mines=None)
    dig(r, c, revealed, mines, h, w) -> hit_mine: bool
        if not in_bounds or already revealed: return
        if is_mine: revealed[r][c] = MINE; return True
        n = count_neighbors(...)
        revealed[r][c] = ' ' if n == 0 else str(n)
        if n == 0:
            for dr in (-1,0,1):
                for dc in (-1,0,1):
                    if (dr, dc) == (0, 0): continue
                    dig(r+dr, c+dc, ...)
        return False
    has_won(revealed, h, w, num_mines) -> bool
    main():
        clear_screen()
        h, w, m = get_board_settings()
        mines = place_mines(h, w, m)
        revealed = [[HIDDEN]*w for _ in range(h)]
        while True:
            display_board(...)
            col = prompt_int(0, w-1, "How many over...")
            row = prompt_int(0, h-1, "How many down...")
            if revealed[row][col] is already revealed:
                print "That cell already exposed! Try again"
                continue
            if dig(...):  # hit a mine
                display_board(reveal_all=True)
                print "Uh Oh, you hit a mine, You Lost..."
                return
            if has_won(...):
                display_board()
                print "Congratulations! You won."
                return
